# RESEARCH_SCOPE

## 1. 本リポジトリの目的
学習なし（凍結）DINOv2特徴 + 多視点レンダ + 単純統合 + 教師なしクラスタリングで、
比較形態標本の群構造がどこまで回収できるかを検証する。

## 2. やらないこと
- 自前表現学習（再学習・微調整・対照学習など）
- ラベルを使ったモデル選択
- coarse/fine を都合よく拾う score 調整
- hard reject で不都合な結果を除外
- size/texture など補助特徴を追加して性能を“見栄え改善”すること
- 主解析での sweep / best-looking split 探索

## 3. 失敗した場合の解釈
DINOv2 の汎用視覚特徴は粗い形態差には反応しうるが、
比較形態標本のクラスタリングにそのまま十分な表現空間を与えない可能性がある。

## 4. 研究の価値
本研究は精度勝負ではなく、
- どこまで見えるか
- どこから見えないか
- 何を見ているか（ViT注意・近傍・tree）
を明示し、
人間の形質ベース分類と機械の表現ベース分類を比較可能にすることに価値がある。

## 5. 可視化の注意
attention rollout / gradient rollout は「分類形質そのもの」の可視化ではなく、
埋め込み形成に寄与した視覚的手がかりの可視化である。
